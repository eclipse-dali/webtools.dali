Mapping File Wizard

The Mapping File wizard enables you to an orm.xmlfile to a JPA project if no object map exists at the location specified. For example, if you cleared the Create orm.xml option on the JPA Facet page, you can later add the orm.xml file to the src file of the project using this wizard.

The Mapping File Wizard consists of the Mapping File page.
