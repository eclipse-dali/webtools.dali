Configure JAXB Class Generation dialog

Use this dialog to generate JAXB classes from a schema.

Property Description
Source folder Location in which to generate the classes. Click Browse to select a location.
Package Name of the package. Click Browse to select an existing package.
Catalog Name of the catalog file to use to resolve external entity references.
Bindings files Click Add to select the external bindings files
