Mapping an entity

Dali supports the following mapping types for Java persistent entities:

  • Basic mapping

  • Embedded mapping

  • Embedded ID mapping

  • ID mapping

  • Many-to-many mapping

  • Many-to-one mapping

  • One-to-many mapping

  • One-to-one mapping

  • Transient mapping

  • Version mapping
