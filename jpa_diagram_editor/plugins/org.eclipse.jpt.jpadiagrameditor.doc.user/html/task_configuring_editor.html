Configuring the JPA Diagram Editor

Configure the global default settings of the JPA diagram editor

  1. From the main menu, choose Window → Preferences .

  2. In the properties tree, select General → Editors → JPA Diagram Editor .

  3. Specify the JPA diagram editor settings. To restore the global predefined values, choose Restore Defaults.

  4. Choose Apply and then OK.

Configure the JPA diagram editor per project

  1. From the project context menu in the Project Explorer, choose Properties.

  2. Select JPA → JPA Diagram Editor .

  3. Specify the JPA diagram editor settings. To restore the global default settings, choose Restore Defaults.

  4. Choose Apply and then OK.
